Seb's Workshop IT Tools

Tools & Resources

A collection of standalone IT tools grouped by who they're built for. Everything runs in the browser with no installs, no accounts and no data leaving your machine.

🛠️ This is a hobby project. These tools are built in spare time and while every effort is made to ensure they're accurate and functional, I can't guarantee everything works perfectly in every scenario - especially Beta-labelled tools.

💬 Feedback is very welcome. If something doesn't work, gives wrong results, or could be improved, please let me know. This whole collection exists because I wanted better tools for daily engineering work - your input genuinely helps make them better.

💾 Always back up first. Where any tool is used to inform changes to a live system or configuration, please take a backup before making changes. No tool - however well built - is a substitute for a good backup.

Seb's Wireshark Helper

Analyse PCAPs for common issues - TCP retransmissions, DNS failures, DHCP problems, STP loops, Teams/VoIP faults and more. Includes guided troubleshooting scenarios, a capture builder for UniFi and port mirror setups, Wireshark display filter reference, and step-by-step port mirror instructions for DrayTek, Netgear, HP, D-Link and UniFi switches. Everything runs in the browser.

Seb's DrayTek VPN Auditor

Import a DrayTek Remote Dial-in User export to audit every VPN account in one table. Flags weak ciphers, missing 2FA, reused and short passwords, and the firmware export bug that silently corrupts certain characters in passwords. Edit, reorder and re-export back to the router's format.

Import a ConnectWise Manage ticket export for instant reporting. Staleness tracking, SLA overdue and due-soon alerts, engineer workload, company heat, type breakdowns, and a customer-responded queue. Flags zero-hour tickets and unworked opens.
